Info

This page was inspired by https://eternitywall.it/.

Zenon Wall displays messages from the Network of Momentum!
Each entry has a UTC timestamp, the transaction's message, and a link to the transaction.


To participate, send any NoM ZTS token to any address and
include a message starting with "ZW".

Example message: "ZW Hello ZNNAliens!"


Example with znn-cli (available here):
.\znn-cli.exe -u node-address -k keystore send dest-address amount token "message"

Note: Syrius wallet does not support this functionality yet.
